🚀 企业微信推送 3 步配置

第一步：创建企业微信群机器人
1. 打开企业微信，创建一个群聊
2. 群聊右上角 ··· → 群机器人 → 添加机器人
3. 新创建一个机器人，命名为"游戏监控助手"
4. 复制 Webhook 地址（重要！）
   格式：https://qyapi.weixin.qq.com/cgi-bin/webhook/send?key=xxxxx

第二步：添加到 GitHub
1. 进入你的 GitHub 仓库
2. Settings → Secrets and variables → Actions
3. 点击 New repository secret
4. Name 填写：WECOM_WEBHOOK
5. Secret 填写：粘贴刚才复制的 Webhook 地址
6. 点击 Add secret

第三步：测试
1. 推送代码到 GitHub
   git add .
   git commit -m "配置企业微信推送"
   git push

2. Actions → Run workflow 手动触发
3. 等待运行完成
4. 检查企业微信群是否收到消息

✅ 完成！之后每次发现新游戏都会推送到企业微信群

优势：
✓ 完全免费，无消息数量限制
✓ 支持 Markdown 格式，显示美观
✓ 群内所有成员都能收到
✓ 可以点击链接直接访问游戏

详细说明请查看：企业微信配置指南.md
